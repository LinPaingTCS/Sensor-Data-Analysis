# Sensor Data Analysis and Activity Classification

This project loads sensor-style CSV data, performs statistical analysis, visualizes trends, and trains a machine learning model to classify activity states such as rest, walk, and run.

## Features
- Load CSV data with pandas
- Compute mean, min, max, and standard deviation
- Visualize sensor trends with matplotlib
- Train a logistic regression model with scikit-learn
- Predict activity labels for new input data

## Tech Stack
- Python
- pandas
- matplotlib
- scikit-learn

## Example Activities
- rest
- walk
- run
